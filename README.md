# 🎮 Hangman Game (Python)

A simple text-based Hangman game built using Python.

## Features

- Randomly selects a word from a predefined list
- Tracks guessed letters
- Input validation
- Prevents duplicate guesses
- Displays remaining lives
- Win/Lose messages

## Concepts Used

- Functions
- Loops
- Lists
- Strings
- Conditional Statements
- Random Module

## How to Run

```bash
python hangman_game.py
```

## Sample Output

```
_ _ _ _ _ _

Remaining lives: 6

Enter your guess:
```
