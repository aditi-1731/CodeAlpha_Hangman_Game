# 🎮 Hangman Game

![Python](https://img.shields.io/badge/Python-3.x-blue)
![License](https://img.shields.io/badge/License-MIT-green)

A console-based Hangman game built in Python where the player guesses a randomly selected word one letter at a time before running out of lives.

---

## Features

- Random word selection
- Input validation
- Duplicate guess detection
- Remaining lives counter
- Guessed letters tracking
- Win/Lose messages
- Modular function-based design

---

## Tech Stack

- Python 3
- Random Module
- Lists
- Strings
- Functions
- Loops

## How to Run

Clone the repository

```bash
git clone https://github.com/aditi-1731/hangman-game.git
```

Move into the project

```bash
cd hangman-game
```

Run

```bash
python hangman_game.py
```
